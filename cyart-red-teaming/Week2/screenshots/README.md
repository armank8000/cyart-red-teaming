# Screenshots Guide

Add your tool screenshots here using the naming convention below.

## Naming Convention

| File Name | What to Capture |
|-----------|----------------|
| `01_elastic_security_logs.png` | Elastic Security log ingestion view |
| `02_sigma_rule_test.png` | Sigma rule detection firing in terminal |
| `03_remnux_strings_output.png` | REMnux strings command on calc.exe |
| `04_hybrid_analysis_report.png` | Hybrid Analysis sandbox report |
| `05_openvas_scan_results.png` | OpenVAS scan on Metasploitable2 |
| `06_defectdojo_import.png` | DefectDojo vulnerability import screen |
| `07_caldera_attack_path.png` | MITRE Caldera phishing simulation |
| `08_velociraptor_artifacts.png` | Velociraptor process/netstat output |
| `09_suricata_alert.png` | Suricata blocking the malicious IP |
| `10_crowdsec_block.png` | CrowdSec IP block confirmation |
| `11_wazuh_alert.png` | Wazuh alert from capstone exploit |
| `12_metasploit_exploit.png` | Metasploit vsftpd backdoor execution |
| `13_ale_spreadsheet.png` | Google Sheets ALE calculation |
| `14_risk_matrix.png` | 5x5 Risk Matrix screenshot |

## How to Take Screenshots

- **Windows VM:** `Win + Shift + S` → Snip & Sketch
- **Linux VM:** `scrot filename.png` or `gnome-screenshot`
- **Kali/REMnux:** `screenshot -f filename.png`
